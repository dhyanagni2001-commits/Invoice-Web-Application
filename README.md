## Working
1. User logs in securely  
2. User enters customer and invoice details  
3. System calculates totals and taxes  
4. Invoice is generated dynamically  
5. **Invoice is exported as a PDF**  
6. Records are stored securely in the database  

---

## Setup
1. Install XAMPP 
2. Place project in `htdocs`  
3. Import database into MySQL  
4. Start Apache & MySQL  
5. Open `http://localhost/invoice-app`
